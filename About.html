Tentang Proyek Ini

Sebuah perjalanan dalam mempelajari pengembangan web dengan API.

Dibuat oleh Muhammad Habib

Selamat datang di KlikFilm! Proyek ini saya buat sebagai sarana untuk mendalami interaksi dengan API eksternal menggunakan JavaScript modern. Tujuannya adalah menciptakan sebuah aplikasi web yang fungsional, responsif, dan menyenangkan untuk digunakan.

Teknologi yang Digunakan

HTML5
CSS3
JavaScript
Bootstrap 5
TMDB API

Fitur Utama

  • Pencarian Film & TV Series
  • Tampilan Film & TV Populer
  • Detail Lengkap (Sinopsis, Aktor, Rating, dll.)
  • Tombol untuk Menonton Trailer
  • Desain Sepenuhnya Responsif
